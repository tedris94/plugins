Educare import system add-ons.

## == Description ==

Educare Import System allow user to import data from Excel files, with the extensions of (.xlsx, .xlsm or .xls). Also, it will help to create demo file for import system.

## == Installation ==

1. Download `Educare import system` from official website of FixBD (https://fixbd.net)
2. Then, in your WordPress administration panel, go to Plugins > Add New and click the Upload Plugin button at the top of the page. (For manualy installation, Upload `educare-import-system.zip` to the `/wp-content/plugins/` directory.)
3. Activate the plugin (Educare import system) through the 'Plugins' menu in WordPress.
4. Make sure Educare premium version is active.
5. Then, Go to "Educare > All Students||Teachers||Results > click import tab for use the plugin.
