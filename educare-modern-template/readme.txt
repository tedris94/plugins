Educare modern template add-ons.

## == Description ==

Select our Modern Results Card for a timeless data presentation, simplifying information clarity and facilitating informed decision-making processes.

## == Installation ==

1. Download `Educare Modern Template` from official website of FixBD (https://fixbd.net)
2. Then, in your WordPress administration panel, go to Plugins > Add New and click the Upload Plugin button at the top of the page. (For manualy installation, Upload `educare-modern-template.zip` to the `/wp-content/plugins/` directory.)
3. Activate the plugin (Educare Modern Template) through the 'Plugins' menu in WordPress.
4. Make sure Educare premium version is active.
5. Then, Go to "Educare > Settings > Results Card for active and configure the plugin.
