Custom report card template addon for Educare plugin with advanced grading system and professional layout design.

== Description ==

Educare Custom Template is a powerful addon for the Educare plugin that provides a professional, customizable report card template with advanced grading features. This plugin enhances the standard Educare report cards with a modern design and flexible grading system.

= Key Features =

* **Professional Layout**: Clean, modern report card design suitable for academic institutions
* **Advanced Grading System**: Flexible grading key configuration with customizable grade boundaries
* **Multiple Template Formats**: Support for various report card layouts and styles
* **Easy Integration**: Seamless integration with the main Educare plugin
* **Customizable Design**: Modify colors, fonts, and layout elements to match your institution's branding
* **Print-Ready Output**: Optimized for both digital viewing and professional printing

= Requirements =

This plugin requires the main Educare plugin to be installed and activated. It extends the functionality of Educare by providing additional template options for report card generation.

= Template Features =

* Student information section with photo support
* Subject-wise marks and grades display
* Term-wise performance tracking
* Class statistics and rankings
* Principal and class teacher signature areas
* School branding and logo integration
* Responsive design for various screen sizes

== Installation ==

1. Ensure the main Educare plugin is installed and activated
2. Upload the plugin files to the `/wp-content/plugins/educare-custom-template` directory, or install the plugin through the WordPress plugins screen directly
3. Activate the plugin through the 'Plugins' screen in WordPress
4. Navigate to the Educare settings to configure the custom template options
5. Select the custom template from the available template options when generating report cards

== Frequently Asked Questions ==

= Does this plugin work without the main Educare plugin? =

No, this is an addon plugin that requires the main Educare plugin to be installed and activated. It extends the functionality of Educare by providing additional template options.

= Can I customize the template design? =

Yes, the plugin provides various customization options through the settings panel. You can modify colors, layouts, and other design elements to match your institution's requirements.

= Is the template print-ready? =

Yes, the template is optimized for both digital viewing and professional printing with appropriate page breaks and formatting.

= Can I use this with multiple schools or institutions? =

Yes, the template can be customized for different institutions through the settings panel, allowing you to configure branding elements, logos, and other institution-specific information.
