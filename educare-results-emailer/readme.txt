Professional email automation addon for Educare plugin. Send student results to parents via email with PDF attachments and bulk email capabilities.

== Description ==

Educare Results Emailer is a comprehensive email automation addon for the Educare plugin that enables schools and educational institutions to automatically send student results to parents and guardians via email. The plugin generates professional PDF reports and delivers them through a robust email system with advanced features.

= Key Features =

* **Automated Email Delivery**: Send individual or bulk emails to parents with student results
* **PDF Report Generation**: Create professional PDF reports with customizable templates
* **SMTP Configuration**: Advanced SMTP settings with support for popular email services
* **Bulk Email Management**: Send results to entire classes or selected student groups
* **Email Templates**: Customizable email templates with merge tags for personalization
* **Delivery Tracking**: Monitor email delivery status and handle failures gracefully
* **Role-based Access**: Configurable access control for different user roles
* **Multi-template Support**: Support for various report card templates and formats

= Email Features =

* Professional email templates with school branding
* Attachment management for PDF reports
* Email scheduling and queue management
* Bounce handling and delivery notifications
* Parent contact management
* Email history and logging

= PDF Generation =

* High-quality PDF report generation
* Multiple template options
* Custom styling and branding
* Print-ready formatting
* Batch PDF creation for bulk operations

= SMTP Integration =

* Support for major email providers (Gmail, Outlook, Brevo, etc.)
* SSL/TLS encryption support
* Authentication management
* Connection testing and diagnostics
* Error handling and retry mechanisms

== Installation ==

1. Ensure the main Educare plugin is installed and activated
2. Upload the plugin files to the `/wp-content/plugins/educare-results-emailer` directory, or install the plugin through the WordPress plugins screen directly
3. Activate the plugin through the 'Plugins' screen in WordPress
4. Navigate to the Educare Results Emailer settings to configure SMTP and email options
5. Configure your email templates and sender information
6. Test the email functionality before sending bulk emails

= Initial Setup =

1. **SMTP Configuration**: Set up your email server settings in the SMTP configuration section
2. **Email Templates**: Customize the email templates with your school's branding and messaging
3. **User Permissions**: Configure which user roles can access the email functionality
4. **Test Emails**: Send test emails to verify the configuration is working correctly

== Frequently Asked Questions ==

= Does this plugin work without the main Educare plugin? =

No, this is an addon plugin that requires the main Educare plugin to be installed and activated. It extends Educare's functionality by adding email automation capabilities.

= What email providers are supported? =

The plugin supports any SMTP-enabled email provider including Gmail, Outlook, Yahoo, Brevo (formerly Sendinblue), Mailgun, and custom SMTP servers.

= Can I customize the email templates? =

Yes, the plugin includes a template editor where you can customize email content, styling, and merge tags to personalize messages for each recipient.

= Is there a limit on bulk emails? =

The plugin respects your email provider's sending limits. It includes queue management to handle large volumes of emails without overwhelming your SMTP server.

= What PDF libraries are used? =

The plugin uses industry-standard PDF generation libraries (note: for WordPress.org compliance, these may need approval or alternative implementation).

= Can I track email delivery? =

Yes, the plugin includes delivery tracking and logging features to monitor successful deliveries and handle failures appropriately.
